Place your image files in this directory.

Supported formats:
- PNG (.png)
- JPEG (.jpg, .jpeg)
- GIF (.gif)
- WebP (.webp)

The application will automatically detect and load all images from this folder.

For best results:
- Use clear, high-quality images
- Keep filenames simple and descriptive
- Recommended size: at least 500x500 pixels

Images will be automatically set to "Active" when first detected.
Use the Admin panel to toggle images on/off for polls.
